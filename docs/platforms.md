# Platforms

- iOS: SwiftUI app with future Network Extension Packet Tunnel Provider.
- Android: Kotlin/Compose app with future `VpnService` and foreground service integration.
- macOS: SwiftUI app with future Network Extension or helper communication.
- Windows: WinUI 3 app with future Windows Service and Wintun integration.
- Linux: GTK4/libadwaita app with future systemd, TUN, firewall, and resolver integration.
- Desktop CLI and daemon: Rust binaries for Windows, macOS, and Linux.
