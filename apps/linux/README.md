# Linux App

GTK4 + libadwaita Rust skeleton. Locales are limited to `en-AU`, `zh-Hans`, and `en-XA`.

Install dependencies on Debian/Ubuntu-like systems:

```sh
sudo apt install libgtk-4-dev libadwaita-1-dev
```

Future work will package the app and connect it to `atlas-daemon` over a Unix domain socket.
